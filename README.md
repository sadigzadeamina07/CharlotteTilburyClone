## 🇺🇸 English

### ✦ About the Project

A high-end, **Charlotte Tilbury**-inspired frontend for a luxury beauty web application. Features a responsive, aesthetic design optimised for an elegant shopping experience — crafted with industry-standard web technologies for optimal performance.

### ⟡ Features

| | Feature | Description |
|--|---------|-------------|
| ✦ | **Premium UI / UX** | Sophisticated design with seamless sliders and fluid animations via Swiper |
| ⟡ | **Categorization** | Structured sections for makeup, skincare, and foundation products |
| ♡ | **Wishlist** | Interactive product saving via heart icons |
| ◇ | **Modern Architecture** | Industry-standard tech stack for speed and reliability |

### ◇ Technologies

`React v19` · `Tailwind CSS` · `Vite` · `Swiper` · `Axios`

### ♡ Getting Started

**1. Clone the repository**
```bash
git clone <repo-url>
```

**2. Install dependencies**
```bash
npm install
```

**3. Start the development server**
```bash
npm run dev
```
